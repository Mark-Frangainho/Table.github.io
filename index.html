Understanding Table syntx for HTML

Tag Name: Description
<Table> Table To create a table
<Thead> Table head To organize Table headings
<Tbody> Table body To Organize table data within a table
<Tr> Table Row To create a rows in a table
<Th> Table heading To create a bolded table heading
<Td> Table Data To create table data underneath table heading
<Tfoot> Table footer To create table footer container
